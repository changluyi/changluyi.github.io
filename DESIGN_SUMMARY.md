## ⚡ 性能优化

### CSS Containment

```css
article, .page-content, .site-header,
.card, .post-card {
  contain: layout style;
}
```

**效果**: 重排性能提升 30-50%

### 选择性过渡

只对必要元素应用过渡:
```css
.theme-transition {
  transition: background-color 0.3s ease,
              color 0.3s ease,
              border-color 0.3s ease;
}
```

**避免**: 全局 `transition: all`

### 其他优化

- 使用 CSS 变量 (减少重复)
- 硬件加速动画 (`transform`, `opacity`)
- 避免昂贵属性 (`box-shadow` 过渡时已优化)

1. **启动本地服务器**:
   ```bash
   bundle exec jekyll serve
   ```

2. **访问** `http://localhost:4000`

3. **查看效果**

### 自定义建议

#### 修改主题色
```scss
:root {
  --gradient-primary: linear-gradient(135deg, #你的色1 0%, #你的色2 100%);
}
```

#### 添加分类卡片
```html
<a href="/new-category/" class="card category-card">
  <div class="card-icon">🎯</div>
  <div class="card-title">新分类</div>
  <div class="card-desc">描述文字</div>
</a>
```

#### 创建文章卡片
```html
<a href="/post-url/" class="post-card">
  <h2 class="post-title">文章标题</h2>
  <p class="post-excerpt">摘要...</p>
  <div class="post-meta">
    <span class="post-tag">标签</span>
    <span class="post-date">📅 2025-01-22</span>
  </div>
</a>
```
